# Geo AI Workflow

This project is an automation workflow built using **n8n** for extracting and processing data from a website. The workflow is designed to scrape, clean, and structure the data for further use in an AI-based system. This is part of the **AI Workflow** in the **Geo Project**.

## Features

- **Data scraping**: Pulls data from a specified URL (e.g., Chaos website).
- **Data cleaning**: Removes unnecessary HTML tags and other noise.
- **Workflow Automation**: Built using n8n, a no-code automation tool for integrating various APIs and automating repetitive tasks.
- **Integration with OpenAI**: AI-powered model (e.g., GPT) can be integrated to generate content based on the cleaned data.

## Installation

### Requirements

- [n8n](https://n8n.io/) - An open-source workflow automation tool.
- Node.js
- Git
